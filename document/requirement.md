# 减法人生软件功能设计

2.1.1. 首页 (HomePage)
核心元素：
• 简约的水墨背景
• 禅意环形进度指示器 (75%完成度)
• 今日格言卡片
• 待减项目清单
• 半透明导航栏

2.1.2. 减法清单 (ListPage)
特色功能：
• 卷轴式项目展示
• 分类标签系统 (物质、数字、时间)
• 进度条可视化
• 优雅的添加按钮

2.1.3. 习惯培养 (HabitsPage)
关键特点：
• 竹简纹理背景
• 周进度概览 (5/7)
• 习惯打卡时间轴
• 每日任务追踪

2.1.4. 心灵空间 (MindPage)
核心功能：
• 情绪选择器 (平静、愉悦、低落)
• 冥想指导模块
• 今日反思日记
• 心灵锦囊 (每日格言)
